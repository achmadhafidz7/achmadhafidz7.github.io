Introduction

This is my first Data Analysis project. The story behind this is when the world cup 2022 is about to start I am curious about the national team that has the highest rating in FIFA 23 game, and also this is the first month of my journey in learning Data Analysis, so I think to myself "Why not I make this into project?" because I also heard someone said it if you're confused on where to start your Data project, find project that you have passion in, therefore I make this into data project.

The Data

The data used in this project came from multiple sources, that includes :

  1. FIFA 23 Players Dataset - Kaggle
  2. Players that is in the WC squad list - Wikipedia
  3. Players data that doesnt get included in FIFA 23 game - Fifaratings

Objective

The objective of this project is to:

  1. Find Overall rating of every country in World Cup
  2. Create a visualization that can show all country overall rating
  3. Create a visualization that can show all country average age
  4. Find Country with the highest and the lowest Overall Rating
  5. Find Country with the highest DF, MF, and FW overall rating

Chapter 2 - Data Wrangling

Data wrangling involves various operations to transform and prepare data for analysis, it is similar to data cleaning but Data wrangling is much broader task than cleaning, this includes combining datasets and adding values from different sources. In this project I will combine the two .csv we get from before and also adding some values from other sources to fill some of the players data. The result of this process is we will have one table contains all the world cup players without single player missing their values, so we can use it for analysis more accurately.

Load the Data to Excel

Load both .csv files we extracted from python before to Microsoft Excel. In Excel go to Data tab and click Get Data > From File > From text/CSV

INDEX + MATCH

Duplicate the wc_players_wiki sheet and rename it to wc_players_lookup. In this sheet we will fill all the Overall Rating and Age from the Data in fifa23_players_cleaned.

  1. FIrst step is to Find " (captain)" words in the sheet and replace it with "" to remove it.
  2. Create 2 new columns, named 'Age' and 'Overall'
  3. Formula for Age column =INDEX(fifa23_players_cleaned,MATCH(wc_players_lookup!$C13,fifa23_players_cleaned[Full Name],0),3)
  4. Formula for Overall column =INDEX(fifa23_players_cleaned,MATCH(wc_players_lookup!$C2,fifa23_players_cleaned[Full Name],0),4)

After this there will be some rows that are not filled or shown "N.A" value, that is because:

  1. The lookup value not match, the solution is to use known_as tab instead of full name bcs that's what happened in some countries like spain, portugal, or brazil
  2. The name could be different, but they are the same person/player, the solution is to search the name manually and copy the name so it can use the index and match
  3. The player just not exist in FIFA 23 game because not every player will be included in the game, the solution is to go to fifaratings.com and search for the players, for the age just search the player in Google.

Create a master sheet

Create a new sheet and rename it to wc_players_master, and this sheet will contains all the Data we're going to use. Copy all the values from lookup Data and paste it to master sheet. and then, we can start the Data Entry process for filling all the missing values, by doing the solution mentioned before in point 3 above.

After all the players have their age and overall, we can save this workbook and export it for us to use in Tableau.

Chapter 3 - Tableau

In this Chapter we will visualize our data so we can see it more clearly. Imagine looking at thousands of rows of data and try to analyze it straight away, it will take an awful amount of time. So, one of the purpose of Data visualization is to made the analysis easier by looking at different charts and graphs that fit with the analysis. This will allow us to gather insights and explore the data more.

Load the Excel workbook

Load the excel workbook we created before, and choose the wc_players_master.

Overall Rating of all Countries

We're going to make Bar chart that can show all the Countries Overall Rating with conditional formating.

  1. Put the Nationality in Columns and Overall in Rows, and change the measure of Overall from SUM to Average.
  2. Click Label in Marks and check the Show Mark Button to show the number above the bar chart.
  3. Drag Average Overall to Color so we can Color Format it.
  4. Click color in Marks and Edit Color, and change it to Red-Green Diverging
  5. Sort the Chart from highest to lowest(Descending).
  6. Add Continent and Position to Filters, to use it later in dashboard as a filter.
  7. Right click on the AVG(Overall) and Change the Format of the numbers to Number (Custom), change it to 1 Decimal.

Age of all countries

For age bar chart we just need to duplicate the sheet before and make a little bit of change.

  1. Remove the AVG(Overall) and replace it wih AVG(Age).
  2. Change the color of the bar, the lowest will be green and the highest will be brown (Because green match with young and brown match with old just like a plant).

Highest and Lowest overall

For Highest and Lowest chart we will duplicate it just like before.

  1. Put Nationality to Filter to use a filter.
  2. Change the filter setting to Top > By field > Top 10 by Overall Average.
  3. Change the color to dark blue for highest and light blue for lowest.
  4. For the Lowest duplicate the Highest sheet and just change the filter from Top 10 to Bottom 10.
  5. Change the color to dark red for Lowest and light red for highest.

Creating the Dashboard

  1. Set the size of the Dashboard to Fixed Size Custom 1000 x 2000 pixels
  2. To add a filter, first add the first sheet which is overall rating bar chart to the dashboard and then click the sheet, after that the Show Cards icon will be able to be clicked. Choose Filters > Continent & Position.
  3. in the third image I create yellow lines to show different elements in the dashboard, such as Text Box to show text that are seperated from the visualization sheet before, filters, and also the sheet of visualization we created before.
  4. The rest of Dashboard making process is quite easy, we just need to drag the sheet we created before (it's on the left side of the dashboard sheet) and then adjust the size.

Chapter 4 - Result

Dashboard

This is the Dashboard that I created, you can filter the data based on position/continent and also hover over the data to show the detailed info (tooltip).

Insights

  1. Brazil has the highest overall rating of all countries.
  2. Qatar has the lowest overall rating of all countries.
  3. Netherlands and England has the best DF (Defender) Overall Rating, that means this two teams has the best collection of defensive players.
  4. Germany has the highest MF (Midfielder) Overall Rating, that means Germany has the best collection of midfield players.
  5. Argentina has the best FW (Forward) Overall Rating, that means Argentina has the best collection of attacking players.
  6. Iran has the oldest set of players(28.8 Average), while Ghana has the youngest set of players(24.4 Average).
