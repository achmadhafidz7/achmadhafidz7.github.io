Introduction

This is my first Data Analysis project. The story behind this is when the world cup 2022 is about to start I am curious about the national team that has the highest rating in FIFA 23 game, and also this is the first month of my journey in learning Data Analysis, so I think to myself "Why not I make this into project?" because I also heard someone said it if you're confused on where to start your Data project, find project that you have passion in, therefore I started this.

The Data

The data used in this project came from multiple sources, that includes :

  1. FIFA 23 Players Dataset - Kaggle
  2. Players that is in the WC squad list - Wikipedia
  3. Players data that doesnt get included in FIFA 23 game - Fifaratings

Objective

The objective of this project is to:

  1. Find Overall rating of every country in World Cup
  2. Create a visualization that can show all country overall rating
  3. Create a visualization that can show all country average age
  4. Find Country with the highest and the lowest Overall Rating
  5. Find Country with the highest DF, MF, and FW overall rating

Chapter 2 - Data Wrangling

Load the Data to Excel

Load both .csv files we extracted from python before to Microsoft Excel. In Excel go to Data tab and click Get Data > From File > From text/CSV

INDEX + MATCH

Duplicate the wc_players_wiki sheet and rename it to wc_players_lookup. In this sheet we will fill all the Overall Rating and Age from the Data in fifa23_players_cleaned.

  1. FIrst step is to Find " (captain)" words in the sheet and replace it with "" to remove it.
  2. Create 2 new columns, named 'Age' and 'Overall'
  3. Formula for Age column =INDEX(fifa23_players_cleaned,MATCH(wc_players_lookup!$C13,fifa23_players_cleaned[Full Name],0),3)
  4. Formula for Overall column =INDEX(fifa23_players_cleaned,MATCH(wc_players_lookup!$C2,fifa23_players_cleaned[Full Name],0),4)

After this there will be some rows that are not filled or shown "N.A" value, that is because:

  1. The lookup value not match, the solution is to use known_as tab instead of full name bcs that's what happened in some countries like spain, portugal, or brazil
  2. The name could be different, but they are the same person/player, the solution is to search the name manually and copy the name so it can use the index and match
  3. The player just not exist in FIFA 23 game because not every player will be included in the game, the solution is to go to fifaratings.com and search for the players, for the age just search the player in Google.

Create a master sheet

Create a new sheet and rename it to wc_players_master, and this sheet will contains all the Data we're going to use. Copy all the values from lookup Data and paste it to master sheet. and then, we can start the Data Entry process for filling all the missing values, by doing the solution mentioned before.

Insights

  1. Brazil has the highest overall rating of all countries.
  2. Qatar has the lowest overall rating of all countries.
  3. Netherlands and England has the best DF (Defender) Overall Rating, that means this two teams has the best collection of defensive players.
  4. Germany has the highest MF (Midfielder) Overall Rating, that means Germany has the best collection of midfield players.
  5. Argentina has the best FW (Forward) Overall Rating, that means Argentina has the best collection of attacking players.
  6. Iran has the oldest set of players(28.8 Average), while Ghana has the youngest set of players(24.4 Average).
